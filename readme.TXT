Diese Readme dient in erster Linie uns Developern, eine User Readme wird bei Mod Release bereitgestellt.
Jeder der etwas entsprechendes mitzuteilen hat kann diese Readme gerne editieren!



1) Ich habe das Präfix "#C" für das Projekt auf blackwyrmlair reserviert.
Alle Variablen und Dateinamen sollten also im Rahmen dieses Projekts mit "#C" beginnen um die
Kompatibilität zu anderen Mods zu gewährleisten.

2) Alle Dateien sollten UTF-8 decodiert werden damit es bei der Darstellung von Umlauten zu keinen Problemen bei github kommt.
